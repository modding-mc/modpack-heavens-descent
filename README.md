# Heaven's Descent

**Heaven's Descent** is a high-fantasy RPG modpack which starts the action in heaven and descends down as the game progresses:

`Heaven` => `Overworld` => `Nether` => `Deeper Down` => `Ender`

> [!Warning]
> The project is work-in-progress. It is currently undergoing a large refactor & clean-up in order to become compliant with Curseforge publishing guidelines.
